# 三、标准化


1. 归一化
    - 将数据缩放到一个固定范围
    - 常见方式（最小最大归一化）：$X_{norm} = \frac {X-X_{min}} {X_{max}-X_{min}}$
2. 标准化
    - 将数据缩放成均值为0，方差为1的分布
    - 常见方式（Z-Score标准化）：$X_{std} = \frac {X - \mu} \sigma$
    - 标准差$\sigma$，可以用均值绝对偏差$S$替换，对于离群点更鲁棒
        - $S = \frac 1 n (|X_1-\mu| + |X_2-\mu| +... + |X_n-\mu|)$
3. 小数定标
    - 统一缩小$10^n$倍
